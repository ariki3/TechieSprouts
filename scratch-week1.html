Lesson Outline

Introduction

5 minutes

Scratch is a fun way to code games using blocks! Watch the demo above to see the Safari Game. Your goal is to move a lion to catch zebras and score points.

Set Up

5 minutes

  1. Go to scratch.mit.edu and click "Create".
  2. Name your project "Safari Game".
  3. Add a safari backdrop: Click the Stage, then "Choose a Backdrop", and pick "Savanna".
  4. Add sprites: Click "Choose a Sprite", add a Lion and two Zebras. Set Lion size to 50 and Zebras to 30.

Code the Lion

10 minutes

  1. Select the Lion sprite.
  2. Drag these blocks to the Code area:
    • when green flag clicked
    • go to x: 0 y: 0
    • forever
    • if then with change x by 10
    • Repeat for left (change x by -10), up (change y by 10), and down (change y by -10).
  3. Create a "Score" variable (in Variables) and add set [Score] to 0 after when green flag clicked.
  4. Test by clicking the green flag and using arrow keys.

Code the Zebras

10 minutes

  1. Select a Zebra sprite.
  2. Drag these blocks:
    • when green flag clicked
    • show
    • go to random position
    • forever
    • move 5 steps
    • if on edge, bounce
    • if then with:
      • hide
      • change [Score] by 1
      • go to random position
      • show
  3. Copy the code to the other Zebra (drag to its Sprite pane or use the backpack).
  4. Test: Catch zebras with the lion to increase the score.

Add Sounds

5 minutes

  1. For the Lion: Add play sound [Roar] until done after when green flag clicked.
  2. For each Zebra: Add play sound [Pop] until done inside if then, before hide.
  3. Test the sounds during gameplay.
